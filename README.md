# Emotion Classification using Machine Learning

## Objective

This project classifies emotions in text using machine learning techniques.

## Dataset

The dataset contains text comments and their corresponding emotion labels.

## Preprocessing

- Lowercase conversion
- Remove punctuation
- Remove numbers
- Remove URLs
- Tokenization
- Stopword removal

## Feature Extraction

TF-IDF Vectorizer

## Machine Learning Models

- Naive Bayes
- Support Vector Machine (SVM)

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix

## Libraries Used

- pandas
- numpy
- nltk
- scikit-learn
- matplotlib
